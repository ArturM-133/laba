# Лабораторная работа №1: Первичное исследование и оценка качества данных

## Описание проекта
Первичный анализ датасета "Розничные продажи" в рамках дисциплины "Интеллектуальный анализ данных".

**Предметная область:** набор данных содержит информацию о транзакциях, клиентах и товарах, включая демографические признаки и финансовые показатели. Эти данные позволяют анализировать покупательское поведение и финансовые итоги каждой сделки. Возможные задачи интеллектуального анализа данных: прогнозирование спроса, сегментация клиентов по покупательскому поведению и анализ ассоциативных правил для оптимизации ассортимента и кросс-продаж.

  
**Цель лабораторной:** аудит качества данных, построение паспорта датасета, выявление проблем.

## Структура проекта
- `notebooks/01_data_understanding.ipynb` — основной Jupyter Notebook с анализом  
- `report/quality_report.md` — текстовый отчёт с выводами и планом улучшения  
- `data/` — датасет или скрипт загрузки
